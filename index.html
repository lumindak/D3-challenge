Correlations Discovered Between Health Risks and Age, Income and Poverty

This chart shows how the Percentage Obesity (Obese(%)), Percentage Smokers (Smokes(%)) and Percentage without Healthcare (Lacks Healthcare(%)) correlate with the Median Age (Age(Median)), Median Annual Household Income (Annual Household Income) and Percentage in Poverty (In Poverty(%)). Each circle represents a State. The axis parameters can be changed by clicking on axis lablels. The state abbreviations are displayed on the circles. If you move the cursor over the circle you will be able to see the values of the chosen parameters.

Technology Used: Data Driven Documents (D3.js), JavaScript, HTML, CSS, Document Object Models (DOMs), Scalable Vector Graphics (SVG), Bootstrap, jQuery
